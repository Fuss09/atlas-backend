# ATLAS INTELLIGENCE MODEL

Version : 1.0

---

# Introduction

Atlas n'analyse pas les marchés comme un investisseur traditionnel.

La majorité des investisseurs commencent par une entreprise.

Ils regardent ensuite :

- les actualités
- les résultats financiers
- le graphique
- les recommandations

Atlas fonctionne à l'inverse.

Il commence par comprendre le monde.

Les entreprises ne sont que la conséquence des changements économiques, technologiques et géopolitiques.

Notre objectif est donc d'identifier les changements avant qu'ils soient pleinement intégrés par le marché.

---

# Les Mégatendances

Une mégatendance représente un changement majeur qui s'étale sur plusieurs années.

Exemples :

- Intelligence Artificielle
- Robotique
- Quantique
- Spatial
- Cybersécurité
- Transition énergétique
- Nucléaire
- Biotechnologies
- GLP-1
- Véhicules autonomes
- Défense
- Informatique haute performance
- Cloud Computing

Une mégatendance possède :

- un niveau de maturité
- un rythme de croissance
- un niveau de risque
- une liste de technologies associées
- une liste d'entreprises concernées

# Les Signaux Faibles

La force d'Atlas est d'identifier des événements qui, pris individuellement, semblent peu importants.

Exemple :

Une startup recrute 200 ingénieurs.

↓

Quelques semaines plus tard, elle dépose plusieurs brevets.

↓

Puis Microsoft signe un partenariat.

↓

Puis la FDA accélère une procédure.

Pris séparément, ces événements semblent anodins.

Ensemble, ils constituent un signal fort.

Atlas doit être capable de détecter ces corrélations.

# Les recommandations

A surveiller

Neutre

---

# Les alertes

Une alerte ne sera envoyée que lorsqu'un événement modifie réellement l'analyse.

Exemple :

Le score passe de 62 à 84.

Un brevet stratégique est publié.

La FDA valide un traitement.

Un dirigeant achète massivement des actions.

Une entreprise annonce un partenariat majeur.

Atlas privilégie la qualité des alertes à leur quantité.

# Principe fondamental

Atlas ne cherche pas à prédire le futur.

Atlas cherche à détecter les changements avant qu'ils deviennent évidents.

C'est cette différence qui constitue son avantage.

---

# Vision finale

L'objectif n'est pas de construire un scanner de news.

L'objectif est de construire le meilleur moteur d'intelligence d'investissement pour les particuliers.

Chaque recommandation devra être :

Compréhensible.

Explicable.

Traçable.

Actionnable.
